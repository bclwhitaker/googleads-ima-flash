------------------------------------------------------------
Running the sample
------------------------------------------------------------

  1.3.1 Generic Instructions
  --------------------------

  a. Extract the zipped file to a convenient location on your computer.

  b. We recommend that you place the samples in an accessible web server
     directory so that you can view the HTML pages through your browser, but
     this isn't required.

  c. Now, you can run the samples either by accessing the index.html pages
     directly from your browser, or by opening the .as and .mxml files in your
     Flash IDE, installing the SDK, and running them in the IDE.

  1.3.2 Interacting with the sample page
  -----------------------------------

  a. Load the index.html file in a browser.

  b. Once on the demo page, a sample content video will wait to be played.

  c. Select one of the radio buttons above the player to request and play either
     a linear or non-linear ad.

  d. Alternatively, press the "play" button to start the content.
